# Terminal Cheat Sheet

## What is it good for?

The terminals is your main working tool. This is where you'll imput
all of your lines of code in order to get results

## Shortcuts

```html
<table>
    <thead>
        <tr>
            <th>Shortcut</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td><code>Ctrl + A</code></td>
            <td>Move to the start of the line.</td>
        </tr>
        <tr>
            <td><code>Ctrl + E</code></td>
            <td>Move to the end of the line.</td>
        </tr>
        <tr>
            <td><code>Ctrl + L</code></td>
            <td>Clear the terminal screen.</td>
        </tr>
        <tr>
            <td><code>Ctrl + C</code></td>
            <td>Stop the current process/command.</td>
        </tr>
        <tr>
            <td><code>Ctrl + D</code></td>
            <td>Log out or exit the terminal.</td>
        </tr>
        <tr>
            <td><code>Ctrl + Z</code></td>
            <td>Pause the current process (can be resumed).</td>
        </tr>
        <tr>
            <td><code>Ctrl + R</code></td>
            <td>Search command history (backward search).</td>
        </tr>
        <tr>
            <td><code>Up Arrow</code></td>
            <td>Show the previous command (from the command history).</td>
        </tr>
        <tr>
            <td><code>Down Arrow</code></td>
            <td>Show the next command (from the command history).</td>
        </tr>
        <tr>
            <td><code>Tab</code></td>
            <td>Auto-complete commands, files, or directories.</td>
        </tr>
        <tr>
            <td><code>Tab</code> twice</td>
            <td>List all possible completions.</td>
        </tr>
        <tr>
            <td><code>Ctrl + Shift + C</code></td>
            <td>Copy the selected text or command.</td>
        </tr>
        <tr>
            <td><code>Ctrl + Shift + V</code></td>
            <td>Paste copied text or command.</td>
        </tr>
        <tr>
            <td><code>Ctrl + Shift + N</code></td>
            <td>Open a new terminal window.</td>
        </tr>
        <tr>
            <td><code>Ctrl + Shift + T</code></td>
            <td>Open a new tab in the terminal.</td>
        </tr>
        <tr>
            <td><code>Ctrl + PageUp</code> or <code>Ctrl + PageDown</code></td>
            <td>Switch between terminal tabs.</td>
        </tr>
        <tr>
            <td><code>Alt + Shift + n</code></td>
            <td>Switch to the nth tab</td>
        </tr>
    </tbody>
